# App Store Submission Checklist — Finna

- [ ] App name confirmed with client
- [ ] Bundle ID registered in App Store Connect
- [ ] Apple Developer account enrolled ($99/year)
- [ ] App icon 1024x1024 PNG exported and uploaded
- [ ] Screenshots captured for iPhone 16 Pro Max (6.7"), iPhone 14 Plus (6.5"), iPhone 8 Plus (5.5")
- [ ] App Store description finalized
- [ ] Privacy policy hosted at a public URL
- [ ] Privacy policy URL entered in App Store Connect
- [ ] Age rating questionnaire completed (result: 4+)
- [ ] Category set to Finance
- [ ] Price set to Free
- [ ] Build uploaded via Xcode Organizer or Xcode Cloud
- [ ] TestFlight internal test completed
- [ ] Client has reviewed and approved TestFlight build
- [ ] Submit for Apple Review
